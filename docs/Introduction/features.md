## Features 

### Single GPU support
The algorithms currently supported by Accelerated WEKA are:
- LinearRegression
- LogisticRegression
- Ridge
- Lasso
- ElasticNet
- MBSGDClassifier
- MBSGDRegressor
- MultinomialNB
- BernoulliNB
- GaussianNB
- RandomForestClassifier
- RandomForestRegressor
- SVC
- SVR
- LinearSVC
- KNeighborsRegressor
- KNeighborsClassifier

### Multi-GPU support
The algorithms supported by Accelerated WEKA in multi-GPU mode are:
- KNeighborsRegressor
- KNeighborsClassifier
- LinearRegression
- Ridge
- Lasso
- ElasticNet
- MultinomialNB
- CD
